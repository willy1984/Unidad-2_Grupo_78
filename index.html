INTRODDUCION

A lo largo de la historia han surgido diversos lenguajes de programación web, pero es de saberse que existen algunos que son los más usados debido a su facilidad y versatilidad al momento de desarrollar un proyecto web.

Este trabajo se realizó con el objetivo de compartir conocimiento, informando a la comunidad investigadora los datos esenciales al momento de emprender el diseño de un sitio web. Se empleara html y css, mostrando el trabajo conjunto de ellos y la función que desempeña cada uno en el proyecto, ilustrando las grandes ventajas cuando estos trabajan en equipo.

Además se hará uso de la plataforma github, la cual nos proporciona una manera de mantener nuestros trabajos de programación de una forma más segura, y poder compartirlos a nuestros compañeros, contando con la opción de contribuidores, los cuales pueden agregar archivos al repositorio creado.

OBNJETIVOS

Objetivo general


Crear un sitio web con los requerimientos establecidos, donde cada estudiante debe contribuir de forma directa en la elaboración del OVI en formato HTML5, demostrando el resultado de las investigaciones realizadas obteniendo las estrategias necesarias para el diseño.


Objetivos específicos


• Realizar investigaciones para obtener los requisitos básicos que se deben de tener en cuenta al momento de emprender un proyecto de programación web. • Establecer el diseño más óptimo para un sitio web teniendo en cuenta el uso que se le va dar, donde cada estudiante hará su aporte para llegar a la estética ideal para el proyecto.
• Ejecutar el diseño del OVI en formato html5, cada estudiante se le asignaran tareas para lograr la contribución al trabajo final.

INFORMACION DE LOS PARTICIPANTES
